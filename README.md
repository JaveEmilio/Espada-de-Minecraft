# Espada-de-Minecraft
Este es un programa en Java con POO, que simula el daño de una espada de minecraft. Psdt: tienen un nivel de abstracción alto

# Mejoras
Implemetar Herencia con:
  -Espada de madera
  -Espada de Diamante
Utilizar arreglos para las espadas
